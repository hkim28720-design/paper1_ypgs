# Data Access

This repository contains code for cleaning and analyzing YPGS survey data.

## Raw data

Raw survey files are not guaranteed to be redistributable through this GitHub repository.
Users should obtain the original files from the appropriate source and place them in:

`data/raw/`

## Processed data

Processed files can be regenerated by running the pipeline from the raw data, subject to access rights.

## Supporting documents

Methodological reports and public documentation are stored in:

`docs/GC_YPGS_reports/`

## Important note

Before publishing raw or processed data in this repository, verify that redistribution is permitted by the data provider and the relevant license terms.
